# 🎬 Movie Magic – Smart Ticket Booking Platform

Movie Magic is a modern full-stack movie and event ticket booking platform designed to deliver a fast, elegant, and seamless booking experience. Built using Node.js, Express.js, PostgreSQL (Neon DB), and Vercel, the platform supports real-time bookings, secure authentication, admin management, and a premium responsive UI.

It combines the simplicity of online booking with powerful backend management tools, making it ideal for real-world deployment and academic projects.

## 🌟 Key Highlights

✅ Book Movies, Events, Sports & Shows
✅ Interactive Seat Selection System
✅ Secure User Login / Signup Authentication
✅ Modern Admin Dashboard
✅ Live Database with Neon PostgreSQL
✅ Fully Hosted on Vercel Serverless Infrastructure
✅ Fast, Mobile Responsive & Elegant UI
