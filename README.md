# Movie Magic - Ticket Booking System

**Movie Magic** is a modern, serverless movie ticket booking application built with Node.js, Express, and Neon PostgreSQL. It features a stunning responsive UI, interactive seat booking, and a powerful admin dashboard.

## ✨ Features
- 🎬 **Dynamic Catalog**: Browse the latest movies with trailers and ratings.
- 🎟️ **Seat Selection**: Interactive grid for choosing your favorite seats.
- 🔐 **Secure Auth**: User registration and login powered by `bcrypt`.
- 📊 **Admin Dash**: Full control over movies, theaters, and customer lists.
- ☁️ **Serverless Ready**: Fully optimized for deployment on Vercel.

## 🛠️ Tech Stack
- **Backend**: Node.js, Express (Serverless Functions)
- **Database**: Neon PostgreSQL
- **Frontend**: EJS, Bootstrap 5, Vanilla JS
- **Deployment**: Vercel

## 🚀 Quick Start
1. **Clone the repo**:
   ```bash
   git clone https://github.com/reshmasahukara/movie-magic.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Setup Environment**:
   - Create a `.env` file based on `.env.example`.
   - Add your `DATABASE_URL` from Neon.
4. **Deploy**:
   ```bash
   vercel
   ```
